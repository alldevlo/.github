## 👋 New to the team? Read this first.

**GitHub has two versions of every project: one online, one on your computer.**

Think of GitHub like Google Docs with a manual save button. The online version (github.com/alldevlo) is the **master copy** — it's where everything is backed up and shared with the team. Your local copy (on your computer) is your **working copy** — where you actually make changes.

The key rule: **changes you make locally are invisible to everyone else until you push them to GitHub.** This is not automatic. You have to explicitly save your work online.

**How to always stay safe:**
- Whenever you finish working on something, tell the tool you're using (Claude Code, Cursor, or any AI assistant) to **"push to GitHub"** or **"commit and push"**. It will handle the technical steps.
- If you're unsure whether your work is saved online, just ask: *"Is this pushed to GitHub?"*
- Never just close your laptop and assume it's saved — it isn't until it's pushed.

**How this account is organized:**
All projects live in one place: `github.com/alldevlo`. Each repo has a clear name (`devlo-website`, `devlo-inbox`, etc.) and a description so you always know what you're looking at. On your computer, everything is stored in `Documents/GitHub/` — one folder per project, same name as on GitHub.

When in doubt: **ask Claude Code.** It knows the full setup and can push, update, or explain anything.
